jindo.FileUploader의 특징

jindo.FileUploader의 특징은 다음과 같다.

  • 사용자가 선택할 수 있는 파일 확장자를 지정할 수 있다.
  • 사용자가 파일을 선택하는 것과 동시에 파일을 전송한다.
  • 파일 전송 성공 여부에 따라 예외 처리가 가능하다.
  • 파일은 1개만 선택할 수 있다. 여러 개의 파일을 동시에 선택할 수는 없다.
